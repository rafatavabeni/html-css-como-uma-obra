HTML e CSS como uma obra

Estrutura e acabamento na web

Quando comecei a entender desenvolvimento web, uma comparação deixou tudo mais claro: um site funciona muito como uma obra. Primeiro vem a estrutura. Depois, o acabamento. O HTML organiza a base da página. O CSS transforma essa base em algo visualmente resolvido.

A ideia central

Por que essa analogia funciona tão bem?

Na construção civil, ninguém começa escolhendo a tinta antes de saber onde a parede vai existir. Antes do acabamento, existe a estrutura. Antes da aparência, existe a organização do que está sendo construído. Na web, acontece algo muito parecido: o HTML define o que cada elemento é, enquanto o CSS define como esse elemento aparece.

Isso significa que o HTML trabalha com a base lógica da página: título, parágrafo, imagem, lista, botão e seção. Já o CSS entra depois, cuidando da cor, da tipografia, do espaçamento, do alinhamento e do layout.

Estrutura da web

Planta estrutural de um site

Assim como uma obra começa com um projeto estrutural, um site também possui uma organização base. No HTML, essa estrutura é definida por elementos semânticos que indicam a função de cada parte da página.

Nesse diagrama podemos identificar elementos como header, nav, section, article, aside e footer. Cada um deles representa uma parte da estrutura lógica do site.

  • header – área superior da página, normalmente contém título e identidade do site.
  • nav – bloco de navegação com links importantes.
  • section – divisão temática do conteúdo.
  • article – conteúdo independente, como um texto ou explicação.
  • aside – conteúdo complementar ou lateral.
  • footer – rodapé da página com informações adicionais.

Código explicado

O que cada parte faz

HTML: a estrutura

O HTML marca o que cada coisa é. Ele organiza a página e dá significado ao conteúdo. 

<h1>Meu primeiro site</h1>
<p>Estou aprendendo HTML e CSS.</p>
<img src="imagem.jpg" alt="Mesa com notebook">

CSS: o acabamento

O CSS cuida da apresentação visual. É ele que define cor, fonte, espaçamento e layout. 

h1 {
  color: #5c2d91;
  text-align: center;
}

p {
  line-height: 1.8;
  text-align: justify;
}

Fun fact

Um detalhe histórico importante

O CSS foi proposto em 1994 por Håkon Wium Lie, e a primeira recomendação oficial do W3C saiu em 1996. A ideia era separar estrutura e aparência para deixar a web mais organizada e mais fácil de manter.
