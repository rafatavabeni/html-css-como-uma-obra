Estrutura e acabamento na web

Quando comecei a entender desenvolvimento web, uma comparação deixou tudo claro: um site funciona muito como uma obra. Primeiro vem a estrutura. Depois o acabamento. O HTML organiza a base da página. O CSS transforma essa base em algo visualmente resolvido.

Fun fact histórico

O CSS foi proposto em 1994 por Håkon Wium Lie, engenheiro do CERN. A ideia era separar estrutura e apresentação. Antes disso, todo estilo visual era colocado diretamente no HTML, o que tornava os sites difíceis de manter. Em 1996 o W3C publicou a primeira recomendação oficial do CSS. Essa separação permitiu que a web evoluísse para layouts complexos, design responsivo e sistemas de design modernos.
