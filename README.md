# HTML e CSS como uma obra

Projeto em HTML e CSS que explica a diferença entre estrutura e acabamento na web usando uma analogia com construção civil.

## Ideia do projeto

Na construção civil, primeiro existe a estrutura. Depois vem o acabamento.

Na web, essa lógica aparece de forma muito parecida:

- **HTML** = estrutura da página
- **CSS** = aparência e acabamento visual

O objetivo deste projeto é mostrar isso de forma prática, visual e didática.
