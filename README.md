# HTML e CSS como uma obra

Projeto em HTML e CSS que explica, de forma prática, a diferença entre estrutura e acabamento na web usando uma analogia com construção civil.

## Ideia do projeto

Na construção civil, primeiro existe a estrutura. Depois vem o acabamento.

Na web, essa lógica aparece de forma muito parecida:

- **HTML** = estrutura da página
- **CSS** = aparência e acabamento visual

Para demonstrar isso na prática, este projeto possui duas páginas:

- `index.html` → versão explicativa e estilizada
- `html-puro.html` → versão curta apenas com HTML, sem CSS

## Objetivo

Criar uma explicação simples, visual e didática sobre HTML e CSS, usando o próprio site como exemplo.
